What is GHost Engine?

GHost Engine is a custom real-time 3D graphics engine built from scratch in C++ using the Vulkan API. It started as a learning project and has grown into a fully functional rendering framework covering modern techniques used in production games and simulations.

The engine is open source and hosted on GitHub Pages, where we document our progress, techniques and findings as we build it.

Features

  • Vulkan 1.3 backend
  • Physically Based Rendering
  • Deferred + Forward rendering
  • Image-Based Lighting
  • Shadow mapping (PCF)
  • glTF 2.0 asset loading
  • Async compute passes
  • Descriptor set caching
  • Hot-reload shaders
  • Custom memory allocator
